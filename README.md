# coisas-leves

## Linguagem feita apenas para diversão
## Alguma das funções:
- meMama = soma
- meEsquece = sub
- dividePoPai = divisao
- multiplicaPoPai = multiplicação
- falaCmg = string
- printa = print
- clonaCartao = replicate
- mapzada = map
- naMira = index de uma lista

## Exemplo de uso
```haskell
ghci> meMama 5 5
10

ghci> naMira [1,2,3] 1
Just 2

ghci> crime "1" 1
11
```
